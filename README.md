# XNIL-API

## Install
```cmd
> npm install xnil-api
```

### How to use
```js
const xnilApi = require('xnil-api');

xnilApi.ytdown('https://youtu.be/VIgd99Tes5o?si=cSVPJj6_2PnZOsbW')
    .then(result => {
        console.log(result);
    })
    .catch(error => {
        console.error('Error:', error);
    });
```
